# Sort Visualizer
JavaFX application for demonstration of different sorting algorithms.

Project created in Intellij IDEA.

## Sort algorithms
* Bubble sort
* Insertion sort
* Selection sort
* Cocktail sort
* Quick sort

The app also displays the time of each algorithm, in milliseconds.
